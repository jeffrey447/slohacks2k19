Weight Buddies

A win-win based weight-loss application made by Jason Tran, Jeffrey Ha, and Stanley Lee
at SLO Hacks 2019.

Team up with a partner and a coach in your weight loss journey! Gain virtual currency for reaching your goals and climb up the leaderboard.

  • Login

Virtual Currency Reward System

A win-win based system that doesn't pit partners against each other

  • Partners are rewarded only if both reach their weight goals.
  • If either person reaches a weight checkpoint, it multiplies the total coins they get if they will get in the end.
  • In the future, we can implement a coaching system where well performing users can act as coaches and gain coins through helping partners reach their goal.

Performance Graphing

An interactive chart where partners can view each other's progress

  • Hover your mouse over data points to view date.
  • Choose to report weight any time you want, but failure to do so before checkpoint deadlines means you will gain less coins!
  • In the future, we will implement a verification system with the coach for submitting weight variables.

Leaderboards

  • View people around the world who have made the most progress
  • Contact top players and hire them as coaches.

Full-Fledged Login System

Created using Hashlib and MongoDB. Hashes all passwords and stores it privately on a local server.
